# Blinkit_Sales_Analysis_Dashboard
Analyze Blinkit sales data to identify trends in outlet performance, product categories, and customer ratings.

# Project Overview
This project analyzes Blinkit grocery sales data using Power BI to understand sales performance, product trends, and outlet distribution. The dashboard provides interactive visualizations to help identify high-performing product categories, outlet locations, and customer preferences.

# Key Metrics
Total Sales: 1.20M
Total Items: 9K
Average Sales: 141
Average Rating: 3.92

# Key Analysis
  Sales by Outlet Establishment Year  
  Sales by Item Type
  Sales by Fat Content
  Sales by Outlet Size
  Sales by Outlet Location Tier
  Performance comparison by Outlet Type
